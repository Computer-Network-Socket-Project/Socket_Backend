* 수정 필요

- db_close : mysql 연결을 취소하는 코드가 있는 파일
- db_connect : pymysql을 사용해서 python과 mysql을 연결하는 코드가 있는 파일
- front_connect : flask를 사용해서 json파일로 웹으로 보여주며 front에게 값을 전달하기 위한 파일
- game_over : 게임 종료시 작동하는 함수가 있는 파일
- half : 게임이 후반전으로 바뀔 경우 작동하는 함수가 있는 파일
- home : 기본 화면에서 전체 게임에 대한 정보를 나타내는 코드가 있는 파일
- like : 유저가 좋아요를 누를 경우 그 값이 반영되는 함수가 있는 파일
- post : 관리자가 게시글을 새로 만들 경우, 새로 만든 게시글에 대한 정보가 데이터베이스에 저장되는 코드가 있는 파일
- relay : 유저가 기존 화면(HOME)에서 보고 싶은 게임의 게시글을 클릭해서 해당 페이지에 들어간 경우 그 게임에 대한 정보들을 보여주기 위해 필요한 정보 전달해주기 위한 코드가 있는 파일
- score : 관리자가 게임의 점수를 수정할 경우 작동하는 함수가 있는 파일
